| 家长话术的维度名 | 适合推荐的条件 | 推荐话术要求 |
|----------------|--------------|------------|
| 发现信号 | 孩子的动作或语言表现出情绪明显低落 | 推荐话术要表达出「我看到/注意到你」，回应的起点应该是孩子的异常状态，而非外部任务要求。 |
| 低压力叩门 | 孩子并不想主动表达自己的感受，对话处在初期阶段 | 推荐话术要给予孩子开放选择的空间和权利，让孩子可以无压力地选择「不配合」，家长并不强求回应。 |
| 情绪命名 | 孩子明确表达了情绪状况，希望得到家长共情与理解 | 推荐话术要能准确命名情绪，不带评判，不用问句的形式给孩子承认的压力，不急于解释原因。 |
| 开放式交流 | 孩子变得开放但仍较为沉默，想说一些话但一直停在嘴边 | 推荐话术要问出真正开放的关心问题，不预设答案，不是能被“是”或“否”回答的封闭式问题。 |
| 反映核心感受 | 孩子表达了明确的心情或想法，希望得到家长的认同 | 推荐话术要用自己的话准确复述感受，末尾陈述而非追问，给孩子校准空间。 |
| 承认自身盲区 | 对话基本僵持，孩子说出过一些学校经历或自身感受 | 推荐话术要具体承认自己之前对孩子情绪的忽视，不辩解，承担完不转移，也可以把自己抽离出来，代入孩子的视角想。 |
| 给予肯定 | 孩子在画画或提到想画画，或者孩子表现出明显的自卑情况 | 推荐话术要真诚、具体地肯定，不附条件，肯定完停住。如果孩子最近没提到画画，优先肯定她刚刚展现出的具体行为（比如说出真实感受、敢于表达不舒服），不要硬扯回画画。 |
| 委婉建议 | 当前孩子情绪较为稳定，但不知道做什么 | 推荐话术要提出简单的建议，应该是小、今天、可选、不需要提前改变状态的建议 | 
| 给出选择权 | 当前孩子情绪较为稳定，但孩子较为沉默 | 推荐话术要明确说出「不也可以」，留出空间，且不附带惩罚或失望。 |
